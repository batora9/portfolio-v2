# きっかけ

埼玉大学プログラミングサークル「Maximum」では、活動の1つとして競技プログラミングがあり、部内メンバーは[AtCoder](https://atcoder.jp/)のコンテストに積極的に参加しています。

そこで、メンバーにもっと競争力を持ってもらうために、部内の月間ランキングなんてあったらいいな～と思い開発に取り掛かりました。

# 技術スタック

フロントエンド
- JavaScript (React)

バックエンド
- Node.js (Hono.js)

デプロイ
- Cloudflare Pages
- Cloudflare Workers

# 機能

- 部内メンバーのAtCoderの月間/年間/全期間ランキングを表示
